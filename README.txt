A simple project which demonstrates some basics of Groovy's dynamic dispatch.

The tests at src/test/groovy/demo/JavaExpandoLiteTests.groovy and
src/test/groovy/demo/JavaExpandoLiteTests.groovy describe some of the
relevant behavior.  The implementations being tested there are at
src/main/groovy/demo/GroovyExpandoLite.groovy and
src/main/java/demo/JavaExpandoLite.java.

To run the tests:

./gradlew test
